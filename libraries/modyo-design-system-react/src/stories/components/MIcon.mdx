# Icon

## CSS Variables
<Markdown>
  {`

  | Variable                                 | Type            | Description                           |
  |------------------------------------------|-----------------|---------------------------------------|
  | \`--${PREFIX_BS}m-icon-color\`            | css color unit  | The icon color                        |
  | \`--${PREFIX_BS}m-icon-bg-color\`         | css color unit  | The icon background color             |
  | \`--${PREFIX_BS}m-icon-size\`             | css length unit | The size of icon                      |
  | \`--${PREFIX_BS}m-icon-host-size\`        | css length unit | The size of the host (icon box)       |
  | \`--${PREFIX_BS}m-icon-loading-duration\` | number          | The duration of the loading animation |

  > The colors can ref another css var like a \`var(--${PREFIX_BS}primary)\`

  `}
</Markdown>
