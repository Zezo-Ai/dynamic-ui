# Toast
Toasts use `DToastContainer` and the `useToast` hook which are an abstraction to configure `react-toastify` in conjunction with a DAlert to generate toasts

To understand in more detail the aspects covered by this component, review the following documentation:

+ [React Toastify](https://fkhadra.github.io/react-toastify/introduction/)

<Meta title="Design System/Patterns/Toast" />


## Example of usage

```tsx
import {
  DButton,
  DToastContainer,
  useToast,
} from '@dynamic-framework/ui-react';

export default function UseToastExamples() {
  const { toast } = useToast();
  return (
    <>
      <DButton
        text="Show Toast"
        onClick={() => (
          toast(
            'Example',
            { type: 'info', showClose: true, autoClose: 500 },
          )
        )}
      />
      <DToastContainer position="bottom-center" />
    </>
  );
}
```

## Example of usage with material symbols
To use material symbols it is necessary to configure the DContextProvider

```tsx
import {
  DButton,
  DToastContainer,
  DContextProvider,
  useToast,
} from '@dynamic-framework/ui-react';

export default function UseToastExamples() {
  const { toast } = useToast();
  return (
    <DContextProvider
      icon={{
        materialStyle: true,
        familyClass: 'material-symbols-outlined',
        familyPrefix: '',
      }}
    >
      <DButton
        text="Show Toast"
        onClick={() => (
          toast(
            'Example',
            {
              type: 'info',
              showClose: true,
              autoClose: 500,
              icon: 'info',
              iconClose: 'close',
            },
          )
        )}
      />
      <DToastContainer position="bottom-center" />
    </>
  )
}
```

  </div>
</Canvas>
